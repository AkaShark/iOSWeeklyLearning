# iOSWeeklyLearning
iOS周报仓库，欢迎大家共享内容。

当前内容在`iOSWeeklyLearning_1.md`中。

## 共享形式

当前有两种分享形式

### Issue

可以点击仓库的New issue -> 选择适合自己的模板 -> 填写内容提交即可。

当前有三个模板：bug、阅读、学习资料、

### Pull requests

Pull requests适合熟悉一些markdown语法的同学。

你可以这样共享代码：

1、fork本仓库

2、在当前对应的md文件添加自己的内容

3、rebase远程仓库，如果有冲突的话需要解决冲突

4、提交代码至你的仓库，然后提交Pull requests至该仓库的`main`分支，等待合入即可

之后的提交不再需要fork，仅需234步骤。
